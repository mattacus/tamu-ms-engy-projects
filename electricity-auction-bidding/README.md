# Electricity Auction Bidding Project

A project for my ENGY - 615 Smart Grids class where the objective was to develop a profit-maximizing bidding as an owner of a natural gas generator fleet, for a single auction period.

##

Description:

1.  MA-energy-module-Puller-question-2025: Project prompt and overview

3.  report.pdf: Project summary writeup with charts and tables
